#### 业务开发流程
##### 技术选型：
- 构建工具
1. 构建工具都有哪些？
gulp(流式)、grunt（任务管理），webpack（编译，打包压缩），fis(百度)，prepack(facebook代码优化方案)，rollup(去掉庸俗代码，编译速度快)
2. 为什么要构建工具
资源压缩，静态资源 替换，模块化管理(commonJs,AMD,CMD)，编译处理
3. 用哪些构建工具
webpack(编译)，npm script(任务管理)

- MVVM框架选择

- 模块化设计
* css模块化设计
a. 可复用能继承要完整
b. 周期性迭代

- 自适应方案设计
* 设计方法：
a. 先整体后部分再颗粒化（布局，页面，功能，业务）
b. 先抽象再具体
